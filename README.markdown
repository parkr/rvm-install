# rvm-install

## Headache-free RVM installer

### Compatibility

- Mac OS X
- Most linux systems (Ubuntu, CentOS, etc.)

### Installation Process

    \curl -L https://raw.github.com/parkr/rvm-install/master/rvm-install.sh | bash
